# game-portfolio

コーディング パートナー
「Tiled」を使って本格的なマップ作りに挑戦するルートBですね！素晴らしい決断です。 これができれば、ポートフォリオのクオリティは一気に「プロの作品」レベルに引き上がります。

ご自宅のPCで作業を再開するときのために、**「準備することリスト」と「次に私に送るべき指示（プロンプト）」**をまとめました。

この手順通りに進めれば、迷わず実装までたどり着けます。

【自宅での作業リスト】
1. ツールと素材の準備
まずは「武器」と「材料」を揃えます。

Tiled（タイルド）のインストール

公式サイト (mapeditor.org) から「Tiled」というソフト（無料）をダウンロードしてインストールしてください。

タイルセット（素材）のダウンロード

Tiledでは、バラバラの画像ではなく「たくさんのパーツが1枚にまとまった画像（タイルセット）」を使います。

おすすめ: itch.io で "Modern Interiors" (LimeZu氏) または "Pixel Art Top Down" (Cainos氏) を検索してダウンロードしてください。

ダウンロードした画像（例: interiors.png）を、プロジェクトのフォルダに入れます。

2. Tiledでマップを作る
Tiledを起動して、以下の手順で「理想の美術館」を描いてください。

新しいマップを作成:

設定: 方向「直交 (Orthogonal)」、タイルサイズ「32px」（素材に合わせる）、形式「CSV」

タイルセットを読み込む:

右下の「新しいタイルセット」ボタンから、ダウンロードした画像を読み込みます。

レイヤー（層）を意識して描く（重要！）:

右上のレイヤーパネルで、以下の3つのレイヤーを作って描き分けてください。

Ground (床レイヤー): 床やカーペットを描く。

Walls (壁レイヤー): 壁や、ぶつかる家具を描く。

Decoration (装飾レイヤー): 壁の上の飾りや、花瓶など（おばけより手前に表示したいもの）。

オブジェクトレイヤーを作る:

レイヤーパネルで「オブジェクトレイヤー」を新規作成し、名前を Objects にします。

ここに「四角形」ツールで、**「スタート地点（Spawn）」や「作品の判定エリア（Art）」**を配置します。

※この「オブジェクト」を使って、プログラムでイベント（会話など）を制御します。

3. データの書き出し
マップが完成したら、保存します。

ファイル名: museum.json

形式: JSON マップファイル (*.json)

保存場所: index.html と同じフォルダ

【作業再開時のプロンプト】
マップデータ（museum.json）と素材画像（例: tileset.png）がフォルダに入った状態で、以下のテキストを私に送ってください。

これを送ってくれれば、私は**「Tiledで作ったマップをPhaserで読み込んで、壁の当たり判定や作品のイベントを自動で設定するコード」**を作成して渡すことができます。

【依頼】Tiledで作ったマップを実装したい

自宅で「Tiled」を使ってマップを作成しました。以下の構成になっています。

1. 用意したファイル

マップデータ: museum.json

タイルセット画像: tileset.png (※もし違う名前なら書き換えてください)

2. Tiledのレイヤー構成

Ground (床)

Walls (壁・障害物)

Decoration (装飾・おばけより手前)

Objects (オブジェクトレイヤー)
